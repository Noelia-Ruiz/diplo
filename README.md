# Diplomatura Universitaria en Control de Calidad

Este repositorio contiene ejercicios, ejemplos y materiales desarrollados durante la cursada de la Diplomatura Universitaria en Control de Calidad.

## Estructura del repositorio

- **clase_3, clase_4, clase_5, ...**: Carpetas con ejercicios prácticos de cada clase.
- **optativos/**: Ejercicios y materiales optativos.
- **hello.py**: Ejemplo introductorio.

Cada carpeta de clase contiene subcarpetas o archivos con ejercicios temáticos (condicionales, ciclos, listas, tuplas, sets, diccionarios, etc.).

## Objetivo

El objetivo de este repositorio es documentar el avance y los trabajos prácticos realizados a lo largo de la diplomatura, facilitando el repaso y la consulta de conceptos clave de programación y control de calidad.

## Notas

- El repositorio irá creciendo a medida que avance la cursada.
- Los ejercicios están escritos en Python y cubren desde conceptos básicos hasta temas más avanzados.
- Se recomienda clonar este repositorio y probar los ejercicios localmente.

## Cómo usar este repositorio

1. Clona el repositorio:
   ```
   git clone <https://github.com/Noelia-Ruiz/diplo>
   ```
2. Ingresa a la carpeta del repositorio:
   ```
   cd diplo
   ```
3. Ejecuta los ejercicios con Python:
   ```
   python ruta/al/archivo.py
   ```
